# Frontend

Next.js frontend for the Ethara assignment.

## Stack

- Next.js
- React
- TypeScript
- shadcn/ui

## Run locally

```bash
cd frontend
npm install
```

Create `frontend/.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

Start the app:

```bash
npm run dev
```

Runs on `http://localhost:3000`.
